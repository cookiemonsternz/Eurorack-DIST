## Distortion

We start with some basic input filtering, same as in the OD circuit, just a highpass filter set at 7Hz and some scaling down to typical guitar-ish level input ranges instead of the eurorack +-5V:

![input stage](https://cdn.hackclub.com/019e34a7-eee6-700a-bdfa-01991b8e5b11/paste-1778999879548.png)

The reference circuit then had a filter set at 172kHz, which also helps to prevent overcurrent to the op amp. I've subbed this out for one at a more reasonable 32kHz, just to remove all energy we don't need from the signal, while still keeping the overcurrent protection.

![Filter](https://cdn.hackclub.com/019e34ab-b34c-7cca-9f9a-55f8d7684ec2/paste-1779000127426.png)

Next up, the clipping stage, which is rather similar to the overdrive's clipping stage. The main difference is the clipping diodes shunt to VSS, instead or being placed in the op-amp feedback path. This is generally the main difference between overdrive and distortion. Overdrive will clip softly, rounding out sharp edges, while distortions clip sharply, chopping off bits of the waveform.

![Clipping section](https://cdn.hackclub.com/019e34b2-fe0d-78b3-97ec-1c457f086a8d/paste-1779000605690.png)

It is worth noting that the original rat used an LM308 op amp, while here I am modelling a TL072. The tl072 op amp has a much faster slew rate than the LM series, so there will be a much greater distribution of higher harmonics. I'll try filtering these out, but if it comes to it I do have some LM series op amps lying about so I can try those as well.

The RAT's tone control is quite literally just a low pass filter, tuned to provide a range of about 375Hz - 37kHz.

![Tone Section](https://cdn.hackclub.com/019e34ba-bb9f-7864-a927-47b239edbb24/paste-1779001112579.png)

And finally some output scaling. The original RAT used a JFET buffer because it used the LM308 op amp, which is too slow to properly buffer audio signals with 20kHz range, but given I'm using the TL072, I can just use this. Unfortunately, I do have to shift the voltage back up to be centered around the bias voltage, but this isn't too big of a deal. Anyways, here is the output buffer and volume control section:

![Output section](https://cdn.hackclub.com/019e34c0-a09c-7b2f-b0a2-c69af8995d01/paste-1779001498944.png)

And that gives us the completed distortion circuit:
![Distortion circuit](https://cdn.hackclub.com/019e34c2-93c7-765d-81d8-547da71b763e/paste-1779001626342.png)

## Fuzz

Next up, a fuzz circuit. This will be modelled off my goat the Big Muff Pi, albeit only roughly.

First up, the input section.
While the real circuit uses a common emmiter amplifier, I'm going to go with an op amp based input circuit, and just chuck some filters on it.

First up the filtering, this is the same case as in the last couple ones, first divide the voltage down to guitar levels, then a ~7Hz high pass and then a low pass, here tuned to around 1kHz (to be roughly accurate to the real circuit).
![Pre stage](https://cdn.hackclub.com/019e3503-4933-7f5f-935d-94b65313f1df/paste-1779005867355.png)

Next up we need amplification of around 10x, which can be achieved with a simple non-inverting amplifier configuration.

![Scaling](https://cdn.hackclub.com/019e3506-8a16-7dad-89b3-693fd6281996/paste-1779006080484.png)

And then some adjustable scaling before the actual clipping section, this is effectively the drive control.
![Drive control](https://cdn.hackclub.com/019e350a-e794-7a4a-a28c-cebc9cbdefbe/paste-1779006366714.png)

For the actual drive circuit, BC239 transistors are used, but these are very similar to the much more readily available BC547 transistors (which I also have on hand) so I'm using those instead.

I was a bit worried that the different voltage would stuff up the transistor stages but from what I can see its fine. I'll think through why later lol. Anyways, heres the first clipping stage:
![Clipping stage A](https://cdn.hackclub.com/019e3515-3be9-721b-8579-27e9c6945fda/paste-1779007043472.png)
This provides a more light distortion, before hard clipping in the next stage, which is an identical circut.

The tone section of this circuit is quite smart, just a low pass and high pass filter being simultaneously controlled by a single knob. Might use this in the future lol.

![Tone control](https://cdn.hackclub.com/019e351d-6093-70d7-8c62-4fe61cb36273/paste-1779007577357.png)

And then to finish it off a basic wee output stage. In the original circuit, they yet again used a common emitter amplifier, and as before, I'll just use an op amp.
After scaling it up, I'll just chuck in a basic volume control as with all the other circuits.
![output](https://cdn.hackclub.com/019e3524-83c3-71b3-ab73-dfc598379139/paste-1779008044972.png)

And with that, this circuit is also done :)
